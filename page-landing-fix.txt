PAGE LANDING FIX ‚Äì SAFE METHOD (NO LAYOUT CHANGES)

Purpose:
Fix extra top gap caused by fixed navbar / sticky bars
by landing the page directly on the main heading.

This method uses ONLY JavaScript scrolling.
No CSS or layout changes required.


==============================
STEP 1 ‚Äì ADD ID TO HEADING
==============================

Example (product_detail.html):

BEFORE:
<h1 class="text-3xl font-extrabold text-gray-900 mb-1">{{ artwork.name }}</h1>

AFTER:
<h1 id="pageTitle"
    class="text-3xl font-extrabold text-gray-900 mb-1">
  {{ artwork.name }}
</h1>


==============================
STEP 2 ‚Äì ADD THIS JS (BOTTOM OF PAGE)
==============================

Place inside {% block extra_js %}

<script>
document.addEventListener("DOMContentLoaded", function () {
  const title = document.getElementById("pageTitle");
  if (!title) return;

  setTimeout(() => {
    const yOffset = -150;   // üîß ADJUST ONLY THIS VALUE
    const y = title.getBoundingClientRect().top
              + window.pageYOffset
              + yOffset;

    window.scrollTo({ top: y, behavior: "auto" });
  }, 150);
});
</script>


==============================
HOW TO ADJUST POSITION
==============================

Negative value = move UP
Positive value = move DOWN

Examples:
-80   ‚Üí slightly up
-120  ‚Üí normal fixed navbar
-150  ‚Üí navbar + category bar (recommended)
-200  ‚Üí more up
+20   ‚Üí move down

ONLY change yOffset. Nothing else.
